Welcome to Racoon, a browser extension for facebook.

This extension allows you to post anonymously on facebook. Other peope with the extension will be able see those posts. The extension also broadcasts your messages beyond your current friend group boundaries.

This is a pre-alpha version! Current features are limited to posting. Posts will expire after 24 hours. Some features that are in development are listed below:

  • Commenting
  • Upvote / Downvote system
  • Collective reputation
  • hashtags

Progress can be tracked via Github
